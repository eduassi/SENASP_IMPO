1. Grid em container fixo

Um grid é um sistema de colunas e linhas desenvolvidos para organizar os conteúdos on-line. No caso deste framework (Bootstrap), cada linha do grid tem no máximo 12 colunas e assim os elementos (textos, imagens e outros recursos) podem ser distribuídos numa linha com soma menor ou igual a 12. Veja um exemplo a seguir. Outrossim, o grid é extremamente importante para o êxito da exibição dos conteúdos nos diversos tamanhos de telas dos dispositivos.

Observação: para efeito de entendimentos ao longo dos textos a seguir, a palavra grid seguida de um número (grid n) significa que a coluna é um container que ocupa n colunas do grid principal de 12 colunas.
